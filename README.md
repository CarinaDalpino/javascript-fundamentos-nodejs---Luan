# 🚀 Funções em JavaScript (Node.js)

Este repositório apresenta exemplos práticos dos principais conceitos de **funções em JavaScript**, utilizando Node.js.

O objetivo é demonstrar desde o uso básico até conceitos mais avançados como **callbacks e promises**, muito utilizados no desenvolvimento moderno.

---
